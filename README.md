# Form&System API
API wrapper for the Form&amp;System CMS

## Config
```php
$config = [
    'url'           => 'http://api.formandsystem.app',
    'version'       => '1',
    'client_id'     => 'your-client-id',
    'client_secret' => 'your-client-secret',
    'cache'         => true,
];
```
